Partner review packet

Evaluate regulated browser automation with concrete workflows, evidence, and handoffs.

This packet helps associations, fintech and compliance reviewers, marketplace maintainers, and ecosystem partners inspect how Komos handles regulated portal work without replacing policy judgment or human review.

Review Checklist

Workflow Boundary

Identify which browser steps are deterministic and which decisions stay with the operations team.

Evidence Package

Confirm that screenshots, downloaded files, portal messages, timestamps, and run logs are returned.

Queue Handoff

Verify that task runs return structured status, exception reason, next action, and reviewer notes.

Integration Fit

Check whether the example can start from n8n, Make, Zapier, Airtable, Vercel, Cloudflare, or an API.
