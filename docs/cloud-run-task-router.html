Cloud Run router

Queue regulated browser automation tasks from Google Cloud Run.

A deployable task router for CRA background screening, FCRA adverse action, insurance eligibility, banking reconciliation, and other regulated portal workflows that need evidence capture.

Back to overview

Use This Pattern When

  • Google Cloud Run owns the public or internal intake endpoint for an operations queue.
  • Komos owns the browser task that logs into portals, downloads evidence, and returns structured outputs.
  • The team needs audit fields, exception reasons, and reviewer handoffs returned to the queue.
