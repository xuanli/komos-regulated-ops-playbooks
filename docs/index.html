Common Contract

The fixtures and schema use a shared request and result shape for regulated task runs. Keep sensitive regulated data in the system of record, send only the fields required for the portal task, and return structured status, evidence URLs, exception reasons, and next actions to the operations queue.

Maintained as public examples for Komos regulated operations automation.
